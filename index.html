Bootstrap Colorpicker 3.0.0 for Twitter Bootstrap

Fancy and customizable colorpicker plugin for Twitter Bootstrap. Originally written by @eyecon and maintained by @itsjaviaguilar and the Github community.

Examples

Simple input field
<input id="cp1" type="text" class="form-control" value="#5367ce" /> <script> $(function() { $('#cp1').colorpicker({ container: $('#cp1').parent() }); }); </script>
As a component
<div id="cp2"></div> <script> $(function() { $('#cp2').colorpicker(); }); </script>
With custom options
Sample overriding the initial color and format
<div id="cp3"></div> <script> $(function() { $('#cp3').colorpicker({ color: '#AA3399', format: 'hsla' }); }); </script>
Working with events
Change background color:
<div id="cp4">Change background color: </div> <script> $(function() { $('#cp4').colorpicker().on('changeColor', function(e) { $('body')[0].style.backgroundColor = e.color.toHex(); }); }); </script>
Transparent color support
<div id="cp5"></div> <script> $(function() { $('#cp5').colorpicker({ color: "transparent", format: "hex" }); }); </script>
Horizontal mode
<div id="cp6"></div> <script> $(function() { $('#cp6').colorpicker({ color: "#88cc33", guideMode: "horizontal" }); }); </script>
Inline mode
<div id="cp7"></div> <style> .inl-bl { display: inline-block; } </style> <script> $(function() { $('#cp7').colorpicker({ color: 'rgba(255,123,123, .33)', className: 'inl-bl' }); }); </script>
Aliased color palette
<div id="cp8"></div> <script> $(function() { $('#cp8').colorpicker({ palette: { 'default': '#777777', 'primary': '#337ab7', 'success': '#5cb85c', 'info': '#5bc0de', 'warning': '#f0ad4e', 'danger': '#d9534f' } }); }); </script>
Customized widget size
Also showing the support of HTML color names
<div id="cp9" data-color="pink"></div> <style> .colorpicker-2x .colorpicker-saturation { width: 200px; height: 200px; } .colorpicker-2x .colorpicker-hue, .colorpicker-2x .colorpicker-alpha { width: 30px; height: 200px; } .colorpicker-2x .colorpicker-color, .colorpicker-2x .colorpicker-color div { height: 30px; } </style> <script> $(function() { $('#cp9').colorpicker({ className: 'colorpicker-2x', guides: { vertical: { saturation: { maxLeft: 200, maxTop: 200 }, hue: { maxTop: 200 }, alpha: { maxTop: 200 } } } }); }); </script>
