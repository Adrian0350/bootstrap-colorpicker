    Playing with changeColor and the Color object

    This example listens to the changeColor event and changes the background color of the parent element using the complementary color of the selected one.
    This also shows how to use the Color object, which extends tinycolor.
    In all events the e event variable contains the colorpicker and color objects, but in the changeColor event it also contains value, representing the color string.
    Code
    <div id="cp1" data-color="rgba(194, 39, 219, 0.4)"></div>
<script>
  $(function () {
    $('#cp1')
      .colorpicker({
        inline: true,
        container: true
      })
      .on('changeColor', function (e) {
        if (!e.colorpicker) {
          // Not yet created
          return;
        }
        e.colorpicker.picker.parents('.panel').find('.panel-heading')
          .css('background-color', e.value);

        e.colorpicker.picker.parents('.panel-body')
          .css('background-color', e.color.complement().toRgbString());
      });
  });
</script>
